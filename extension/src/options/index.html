Available variables & filters

Use {{name}}, optionally with filters, e.g. {{title|slug}} or {{published|date:YYYY-MM-DD}}.

  • {{content}} — the converted Markdown body
  • {{title}}, {{author}}, {{published}}, {{modified}}, {{date}}, {{description}}
  • {{url}}, {{domain}}, {{path}}, {{site}}, {{captured}}, {{today}}, {{time}}
  • {{meta:NAME}} — any page meta tag (e.g. {{meta:og:image}})
  • {{schema:KEY}} — JSON-LD field (e.g. {{schema:datePublished}})
  • {{selector:CSS}} — text of the first match (e.g. {{selector:.byline}})

Filters: lower, upper, trim, slug, default:x, replace:a,b, truncate:120, date:FORMAT.
